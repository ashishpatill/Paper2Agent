---
name: repo-recon
description: Confirm which repository matches the paper, find runnable tutorials or examples, and identify setup blockers before the pipeline runs.
---

Use this skill when the repository is ambiguous or when a paper hints at multiple codebases.

Workflow:
1. Confirm the best repository candidate.
2. Identify tutorial notebooks, example scripts, and install instructions.
3. Call out API keys, heavyweight dependencies, or missing tutorial evidence.
4. Return a clear proceed-or-stop recommendation.
