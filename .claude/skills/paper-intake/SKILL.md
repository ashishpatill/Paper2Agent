---
name: paper-intake
description: Turn a paper URL or PDF into a structured Paper2Agent brief, extract repository hints, and recommend the next workflow step.
---

Use this skill when a paper is the primary input and the repo or implementation path is not yet fully structured.

Workflow:
1. Extract the paper title, short summary, and implementation-oriented capabilities.
2. Identify repository, demo, dataset, and tutorial hints.
3. Recommend the next Paper2Agent step.
4. If repository evidence is weak, hand off to `/repo-recon`.
